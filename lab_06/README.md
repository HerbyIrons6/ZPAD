## Опис проекту
Програма на C++, яка захоплює відеопотік з камери та дозволяє накладати фільтри в реальному часі. Реалізовано модульну архітектуру (обробка клавіш, обробка кадрів).

## Функціонал (керування клавішами)
- `1` — Оригінальне зображення
- `2` — Інверсія кольорів (Invert)
- `3` — Розмиття (Gaussian Blur) — інтенсивність регулюється повзунком
- `4` — Детектор меж (Canny Edge Detection)
- `5` — Ефект зсуву каналів (Glitch)
- `ESC` — Вихід з програми

## Проблеми, що можуть виникнути з Віртуальними машинами
Якщо запускати через ВМ, то скоріш за все, будуть горизонтальні розриви на артефакти кольору. Це відбувається через затримки при
передачі USB-трафіку з хост-системи до ВМ, та через особливості декодування формату YUV/MJPEG через віртуальний USB-міст відповідно.
