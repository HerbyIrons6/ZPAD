# Лабораторна 5: обмін результатами та початковий аналіз

## Опис
Це веб-додаток, створений за допомогою **StreamLit**. Він призначений для початкового аналізу та візуалізації індексів (VCI, TCI, VHI). Програма автоматично генерує набір даних для демонстрації ф-налу.

## Основний ф-нал
- Фільтрація: Вибір конкретного індексу, області, а також налаштування інтервалів тижнів і років за допомогою слайдерів.
- Сортування: Можливість сортування даних за зростанням або спаданням (з вбудованим захистом від конфлікту при виборі обох опцій).
- Візуалізація (на основі Plotly):
  - Вкладка "Дані": Інтерактивна таблиця з відфільтрованими результатами.
  - Вкладка "Динаміка": Лінійний графік часового ряду обраного індексу для вибраної області.
  - Вкладка "Порівняння": Стовпчикова діаграма, яка порівнює середні значення індексу між усіма областями за обраний період (з кольоровим виділенням поточної області).
- Керування станом: Кнопка "Reset" для миттєвого скидання всіх фільтрів до початкових значень та кнопка для генерації нових випадкових даних.

## Інструкція 
1. встановити залежності `pip install -r requirements.txt`
2. запустити веб-додаток у терміналі командою `streamlit run lab_05/app.py` додаток автоматично відкриється в браузері
