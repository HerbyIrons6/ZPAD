## Лабораторна робот №3: Візуалізація даних

## Опис завдання:
У цій лаб роботі проводиться аналз та візуалізація багатовимірного датасету Video Game Sales with Ratings від Kaggle
## Виконано наступне:
- Data Cleaning: очищення пропусків, конвертація типів даних (User_Score у float, і т.п.)
- Побудова графіків залежності між числовими атрибутами (Scatter plot)
- Побудова гістограми розподілу оцінок критиків за 5 заданими діапазонами.
- Багатовимірнa візуалізація
- Побудова ящикового графіка (Boxplot) та теплової карти кореляції (Heatmap).

## Інструкція запуску
1. встановіть необхідні залежності, перейшовши у папку lab_03
   `pip install -r requirements.txt`
2. запустити Jupiter Notebook
3. Відкрити файл `lab3_Visualization.ipynb`
