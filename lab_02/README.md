Лабораторна 2 Наука про дані: підготовчий етап
Робота складалась з двох частин:
1. Автоматизоване завантаження, очищення та аналіз супутникових даних VHI (NOAA) по областях України.
2. Аналіз великого датасету споживання електроенергії (Household Electric Power Consumption) з 
   використанням багатоступеневих фільтрів, профілювання часу, використовуючи %timeit та нормалізації даних

Інструкція запуску:
1. Для запуску потрібно встановити потрібні залежності командою:
    pip install -r requirements.txt
2. далі запустіть jupiter notebook
3. відкрийте на по черзі виконайте все у файлах Part1_NOAA.ipynb та Part2.ipynb
