# Лабораторна 4: візуалізація даних 2

## Опис завдання
Програма для генерації та аналізу гармонік $y(t) = A \cdot \sin(\omega t + \phi)$ з накладеним гаусівським шумом. 
Реалізовано інтерактивний інтерфейс для керування параметрами сигналу та його очищення за допомогою IIR-фільтра Баттерворта.

## Функціонал
- Слайдери для зміни амплітуди, частоти та фази
- Налаштування параметрів шуму (сер.зн. та дисперсія) без перегенерації структури шуму.
- чекбокс для вкл/вимк шуму
- кнопка **reset**

## Інструкція 

1. Встановити залежності: `pip install -r requirements.txt`
2. Запустити скрипт `python lab_04.py`
3. Якщо не будуть працювати повзунки (як у мене спочатку), то натисніть **ctrl + alt + s** => tools => python plots => приберіть галочку в пункті **Show plots in tool window**
